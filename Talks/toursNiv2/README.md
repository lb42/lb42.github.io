# TEI niveau 2
## supports de la formation sur la personnalisation

 - Tous les diapos sont disponibles en format HTML à l'addresse https://lb42.github.io/Talks/toursNiv2/
    - [Personnalisation de la TEI : les principes](https://lb42.github.io/Talks/toursNiv2/Diapos/perso-1.html)
    - [Personnalisation de la TEI : les pratiques](https://lb42.github.io/Talks/toursNiv2/Diapos/perso-2.html)
    - [TEI conformance : implications de la personnalisation](https://lb42.github.io/Talks/toursNiv2/Diapos/perso-3.html)
 - Les diapos utilisent
   - de feuilles de style CSS qui seront recherchés dans le dossier "../css" 
   - de fiches graphiques qui seront recherchés dans le dossier "../media" 
 - Pour faire les TP il faut :
    - avoir une installation recente (24+) d'oXygen
    - copier sur votre machine le dossier "Travaux", contenant des fiches d'exercice associées avec chacun des diapos.
